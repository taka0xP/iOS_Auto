#### 目录结构
```
├── app # 存放app
│   └── NewSkyEyes.app
├── config  # 存放配置文件
├── data    # 存放数据文件
├── demo    # demo文件
├── logs    # log文件
├── providers   # 所有方法
│   ├── account     # 账号相关的
│   ├── all_services    # 全部服务
│   ├── ask_everyone   # 问大家
│   ├── boss        # 老板
│   ├── common      #公共方法
│   ├── company     #公司
│   ├── device      # 设备管理
│   ├── file        # 文件操作
│   ├── login       # 提供log方法
│   ├── mine        # 我的
│   ├── relation    # 关系
│   └── send        # 发送相关
├── report          # html生成报告
└── test_case       # 存放用例
```
